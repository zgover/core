# Site Protection & Error Pages

Control who can see your pages and what visitors see when something's not available.

:::info Plan availability
**Free** for custom error screens; per-screen passwords and maintenance mode may be gated
by tier.
:::

## Per-screen passwords

Protect individual screens with a **password**, so only people with the code can view them.
You can also set a custom **not-found** screen for missing pages.

## Custom error screens

Design your own **404 / 401 / 403 / 503** screens in the Besigner, so error pages match
your brand instead of showing a generic message.

## Maintenance mode

Flip your site into **maintenance mode** to show a designed 503 while you work, then switch
it back when you're ready.

Walkthroughs for setting passwords and wiring up each error screen are on the way.
