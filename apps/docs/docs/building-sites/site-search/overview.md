# Site Search

Give visitors a **search box** to find content across your site, including
[dataset](../../content-and-data/datasets/overview.md) records.

:::info Plan availability
**Paid**. Site search is available on paid tiers.
:::

![The Search component in the elements drawer](/img/besigner/elements-drawer.png)

## How it works

- Add the **search box** to a screen.
- Aglyn provides a built-in **site search page** with a content matcher that searches pages
  and dataset records.

:::note More detailed how-tos coming
Configuration and styling guides are on the way.
:::
