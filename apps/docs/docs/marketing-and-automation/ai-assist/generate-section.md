# AI Generate Section

Where [copy assist](copy-assist.md) writes text, **AI Generate Section** builds **structure**
— a whole section of components — directly on the canvas.

:::info Plan availability
**Paid / env-gated**.
:::

## Use it

1. Describe the section you want (e.g. "a three-column features row with icons and short
   blurbs").
2. Run **AI Generate Section**.
3. Aglyn adds a **constrained subtree** of real components to the canvas, which you then
   refine like anything else.

The output is constrained to valid components and placement, so it drops into your design
cleanly.

## Tips

- Generate a rough section, then adjust with [drag-and-drop](../../building-sites/besigner/drag-drop-hierarchy.md)
  and [inline text editing](../../building-sites/besigner/text-editing.md).
- Combine with the [section & block library](../../building-sites/site-templates/overview.md) for a mix of
  generated and pre-built blocks.
