# Add search to your site

Let visitors find content across your site — including [dataset](../datasets/overview.md)
records — with a **search box**.

:::info Plan availability
**Paid**.
:::

## Steps

1. Add the **search box** component to a screen (or your shared
   [layout](../screens-and-layouts/overview.md#layouts) so it's site-wide).
2. Aglyn provides a **site search page** with a content matcher that searches pages and
   dataset records.
3. Publish.

## Tips

- Put the search box in your header layout so it's reachable from every page.
- Search covers dataset records, so well-structured [datasets](../datasets/overview.md) make
  search more useful.
