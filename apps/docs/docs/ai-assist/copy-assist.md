# Copy assist

**Copy assist** drafts and rewrites text without leaving the Besigner.

:::info Plan availability
**Paid / env-gated** (runs through a managed Claude route).
:::

## Use it

1. Select a text-capable element (a headline, body text, a button label).
2. Invoke **copy assist** on that text prop.
3. Generate fresh copy or rewrite what's there, then accept the result.

It works on **any canvas text prop**, and blog entries can be drafted with AI too.

## Tips

- Give it context in your prompt — the audience, tone, and goal — for better copy.
- Treat output as a first draft; edit inline afterward with
  [inline text editing](../besigner/text-editing.md).
