# Add a locale

Publish your site in another language by adding a **locale variant**.

:::info Plan availability
**Paid**. This is **multilingual v1**.
:::

## Steps

1. Add a **locale** (e.g. `es`, `fr`) to your host.
2. Aglyn creates **locale variants** of your content, sharing the same structure as your
   default language.
3. Translate the screens' text for that locale.
4. Publish. Aglyn emits **hreflang** tags so search engines serve the right language to each
   visitor.

## Tips

- Keep structure identical across locales — translate the words, not the layout.
- Add a [language switcher](language-switcher.md) so visitors can choose their language.
